## Theme

Traffic and condition helpers. NearBell already knows about ETAs
and clock-time arrival; this batch adds the traffic side of the
equation: how much delay there is, how severe it is, what colour
to show, and how to describe it in alert copy.

## Helpers added

- `trafficDelaySeconds` — current vs freeflow ETA delay
- `trafficDelayLabel` — "+5 min delay" copy
- `trafficSeverityTier` — clear / light / moderate / heavy / severe
- `trafficColorForSeverity` — UI colour per tier
- `trafficAdjustedEtaSeconds` — scale ETA by traffic factor
- `trafficFlowDescription` — narrative description per tier
- `trafficIsWorseThan` — has traffic worsened between two tiers?
- `trafficConfidenceLabel` — high / medium / low confidence label
- `trafficIncidentLabel` — incident kind to user-facing label
